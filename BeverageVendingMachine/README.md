Автомат по продаже напитков
ASP.NET Core с использованием MVC 

Веб-сайт имитирует работу автомата по продаже напитков: автомат умеет принимать деньги, выдавать напиток и возвращать сдачу. 
Файл базы в директории App_Data. Попасть в админку сайта, можно добавив в адресной строке "/admin?key=qwe".
Используется Automapper, EF Core
