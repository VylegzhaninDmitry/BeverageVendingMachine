Автомат по продаже напитков
ASP.NET Core с использованием MVC 

Веб-сайт имитирует работу автомата по продаже напитков: автомат умеет принимать деньги, выдавать напиток и возвращать сдачу. 
Файл базы в директории App_Data. Попасть в админку сайта, можно добавив в адресной строке "/admin?key=qwe".

Controllers:
AdminController - набор crud-методов для работы с напитками.
WendingMachineController - набор методов для работы с пользователем

DTOs:

Набор классов для передачи данных

Migrations:

Набор классов миграций

Models:

Набор классов повторяющую таблицу в базе данных

Profiles:

Набор классов для конфигурации маппинга

Views: 

Представление данных пользователю
